# COMANDDOS

- Creamos la rama

` git branch e2 `

- Nos ponemos a trabajar en la rama

` git checkout e2 `

- Añadimos el archivo para que git lo tenga en consideración

`git add e2 `

- Hacemos el commit

` git commit -m "e2" `

- Para ignorar los *.log y *.class, creamos el archivo .gitignore y añadimos:

` *.log *.class `
